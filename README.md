# Space Invaders

A game application where user can fly a ship in space and shot down the enemies with different types of different weapons. The player has three weapons to chose from to shot down the three types of enemies. Each weapon is efective agains specific enemy. The player can use both keyboard and controller to steer his ship. Application contains the the controller panel on the left side of the window. On top of the display the user can verify the connection with the controller. Below the controller conection info, the user can see the chart representing both XY axis of the controller. Additionally on the control panel the user can change from three difficulties (easy, medium and hard). At the bottom of the control panel the user can change the language of the game.

## Key Features

- **Input:** The user's ship can be controlled both by the keyboard and the controller.
- **Graphics:** Application utilizes Qt library with various custom created graphics for ships and projectiles.
- **Difficulty** The user can chose from three levels of difficulty which change the number of starting HP.
